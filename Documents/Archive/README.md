# Archive フォルダについて

このフォルダは、tsumugi Script の **過去の設計資料・旧仕様・歴史的ドキュメント** を保存するための場所です。

現在の実装とは異なる内容が含まれますが、  
以下の理由から削除せずアーカイブとして保持しています：

- 言語設計の経緯を後から参照できる  
- なぜ現在の仕様になったのかを説明できる  
- 将来のリファクタリングや機能追加の際に参考になる  
- 過去の思想や比較対象として価値がある  

## このフォルダに置くべき文書

- 現行仕様とは異なるが、設計思想として価値があるもの  
- 以前検討していたが採用しなかった仕様  
- 設計の歴史を説明する資料  
- 将来の議論の参考になる古いメモ  

## このフォルダに置かないもの

- 現行仕様（Documents に置く）  
- 将来の改善案（Todo に置く）  
- 実装に直接関係する最新の技術資料

これらは現行仕様ではありませんが、  
tsumugi の設計思想を理解する上で重要な資料です。
